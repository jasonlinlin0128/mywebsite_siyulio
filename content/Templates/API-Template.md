`## 請求 (Request)`

`### Headers`

` ```json `

`{`

`"Content-Type": "application/json",`

`"Authorization": "Bearer {token}"`

`}`

` ``` `

`### URL Parameters`

`| 名稱 | 類型 | 必填 | 預設值 | 說明 |`

`|------|------|------|--------|------|`

`| id | string | ✓ | - | 資源 ID |`

`### Query Parameters`

`| 名稱 | 類型 | 必填 | 預設值 | 說明 |`

`|------|------|------|--------|------|`

`| page | number | ✗ | 1 | 頁碼 |`

`| limit | number | ✗ | 20 | 每頁數量 |`

`### Request Body`

` ```json `

`{`

`"field1": "value1",`

`"field2": "value2"`

`}`

` ``` `

`#### Body 欄位說明`

`| 欄位 | 類型 | 必填 | 驗證規則 | 說明 |`

`|------|------|------|----------|------|`

`| field1 | string | ✓ | 長度 3-50 | 欄位說明 |`

`| field2 | number | ✗ | > 0 | 欄位說明 |`

`## 回應 (Response)`

`### 成功回應 (200 OK)`

` ```json `

`{`

`"success": true,`

`"data": {`

`"id": "123",`

`"field1": "value1",`

`"field2": "value2",`

`"createdAt": "2025-11-27T13:00:00Z",`

`"updatedAt": "2025-11-27T13:00:00Z"`

`},`

`"message": "操作成功",`

`"timestamp": "2025-11-27T13:00:00Z"`

`}`

` ``` `

`### 錯誤回應`

`#### 400 Bad Request - 請求參數錯誤`

` ```json `

`{`

`"success": false,`

`"error": {`

`"code": "INVALID_INPUT",`

`"message": "輸入資料格式錯誤",`

`"details": {`

`"field1": "必填欄位"`

`}`

`},`

`"timestamp": "2025-11-27T13:00:00Z"`

`}`

` ``` `

`#### 401 Unauthorized - 未認證`

` ```json `

`{`

`"success": false,`

`"error": {`

`"code": "UNAUTHORIZED",`

`"message": "請先登入"`

`},`

`"timestamp": "2025-11-27T13:00:00Z"`

`}`

` ``` `

`#### 404 Not Found - 資源不存在`

` ```json `

`{`

`"success": false,`

`"error": {`

`"code": "RESOURCE_NOT_FOUND",`

`"message": "找不到指定的資源"`

`},`

`"timestamp": "2025-11-27T13:00:00Z"`

`}`

` ``` `
